Requirements

Before running the programs, ensure that you have:

NVIDIA GPU with CUDA support
CUDA Toolkit installed
GCC/G++ Compiler
Linux Environment or Google Colab
Compilation

Compile a CUDA program using:

nvcc filename.cu -o output
